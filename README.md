# Level 2 - Task 3: Banking System

This project implements a simple console banking system using clean OOP.

## Features
- `BankAccount` class with:
  - `accountNumber`
  - `balance`
- Operations:
  - Deposit
  - Withdraw
  - Check balance
- Edge cases handled:
  - Prevent withdrawing more than the current balance
  - Validate all user input (menu and amounts)

## Project Structure
- `src/Main.java` -> app entry point
- `src/bankingsystem/model/BankAccount.java` -> account model and business logic
- `src/bankingsystem/app/BankingMenu.java` -> menu-based console interaction

## Run
1. Open in IntelliJ.
2. Run `Main`.
